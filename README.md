# 🥚 ShellPet

Raise your terminal pet with your commands.
コマンド履歴を食べて育つ、CLIエンジニアのための育成ゲーム。

## 📖 概要 (Overview)
ShellPetは、あなたのターミナル作業そのものを「ゲーム」に変えるOSSプロダクトです。
Gitを使えば攻撃力が、Dockerを使えば防御力が上がります。
日々のコーディングやサーバー管理が、ペットの成長につながります。

## ✨ 特徴 (Features)
- **Eat Commands**: `git`, `docker`, `vim` などのコマンド履歴を解析してパラメータ化
- **Evolution**: 育成傾向（STR型、INT型など）によって姿が進化（予定）
- **TUI Dashboard**: Go + Bubble Tea によるリッチなターミナルUI
